## 数据特征描述

**集中趋势**

1. 算术平均数(Mean):  
    for $x_i$ in data:  **data_mean** = $(\sum_1^n x_i ) / n $
2. 中位数(Median):  
    for $x_i$ in sorted(data):  **data_median** = $x_{\frac{n}{2}}$ 

3. 众数(Mode)

    出现次数最多的数

4. K百分位数(Percentile)


5. 四分位数(Quartiles)

**离中趋势**

- 极差(Range)
- 方差和标准差
- 离散系数
- 四分位数极差
- 平均绝对离差
